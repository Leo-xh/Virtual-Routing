#### Components:
+ DV routing:
    * Bellman-Ford algorithm
    * Ford-Fulkerson
+ RIP hop count
+ split horizon, route poisoning, holddown
+ RIP version 2: 224.0.0.9 is used to broadcast
+ UDP on port 520
+ Timers
    * Update
    * Invaild
    * Flush
    * Holddown
+ RIPv2 message
    * request
    * response
